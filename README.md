# Projeto Teste para modelo de IA representante da NORYA

> esse modelo é responsável por ser um LLM com o RAG disponivel da startup, servindo apenas de teste e capacitação para em breve dar inicio a estruturação do modelo oficial 

> o que esse repositório contém?

> esse repositório contém o código respondesavel por fazer o ajuste de um modelo treinado sem o uso de uma API, ou seja, sem as ferramentas prontas, estamos usando um sistema de incorporação de dados ( RAG ) para fazer a adiçao de extra dados cujo quais o modelo deve usar para gerar as respostas tendo como base a visão da NORYA
